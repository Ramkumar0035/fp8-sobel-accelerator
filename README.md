# FP8 Sobel Edge Detection Accelerator

## Overview
This project implements a Sobel edge detection accelerator using FP8 and FP16 formats in Verilog. The focus is on reducing data width and improving hardware efficiency.

## Research Status
This work is part of an ongoing research project and is currently under review. Detailed implementation is limited due to publication considerations.

## Features
- FP8 and FP16 computation
- Reduced data width (compared to FP32)
- Hardware-based Sobel filtering

## Results
- SSIM: 0.82
- PSNR: 37 dB

## Note
Some implementation details are intentionally abstracted due to research publication policies.
